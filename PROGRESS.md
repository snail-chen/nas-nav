# 项目进度文档

## 2025-11-30

### 功能更新
- **UI 重构**: 将整体风格调整为 macOS 16 桌面风格。
  - 实现了顶部状态栏 (Top Bar)。
  - 实现了底部动态 Dock 栏 (使用 Framer Motion 实现放大效果)。
  - 实现了 Launchpad 风格的应用网格布局。
  - 实现了 macOS 风格的窗口模态框 (红绿灯按钮、磨砂玻璃效果)。

- **视觉优化**:
  - **配色方案**: 更新为“深邃星空蓝” (Deep Space Blue) 主题。
    - 背景：深蓝/黑渐变。
    - 图标：青色/蓝色渐变，带微光效果。
    - 组件：半透明磨砂玻璃质感，搭配青色边框高亮。
  - **粒子特效**: 更新了粒子颜色（青、蓝、紫、白）和运动轨迹，模拟星空效果。
  - **滚动条**: 自定义了 Webkit 滚动条样式，适配深色主题。

### 技术栈
- React + Vite
- Tailwind CSS
- Framer Motion (动画)
- Lucide React (图标)
